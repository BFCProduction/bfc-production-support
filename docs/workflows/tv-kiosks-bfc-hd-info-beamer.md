# TV Kiosks - BFC HD (Info Beamer)

## Overview

| | |
|---|---|
| **System** | Info Beamer |
| **Playlist** | BFC HD |
| **Asset Specs** | 1920 x 1080 PNG/JPG |
| **Dropbox Folder** | /production/pro presenter/TV Channels/BFC HD |

---

## Prerequisites

- [ ] Export slide(s) at exactly 1920 x 1080
- [ ] Save as PNG or JPG
- [ ] Name each file: `YYYY.MM.DD - Slide Title` (example: `2026.02.23 - Student Night`)
- [ ] Save to Dropbox: `/production/pro presenter/TV Channels/BFC HD`
- [ ] Confirm content is final (spelling, dates/times, branding)

---

## Procedure

- [ ] Log into [info-beamer.com](https://info-beamer.com) — see credentials in password manager
- [ ] Assets: drag new image(s) into the upload drop area; confirm they appear in Assets
- [ ] Playlists: Edit the BFC HD playlist
- [ ] Bottom of playlist: **+ Add Assets** → select asset(s) → **Add to playlist**
- [ ] Reorder slides if needed
- [ ] **Save and Apply Playlist**; wait for confirmation

---

## Verification

- [ ] Check at least one kiosk (or preview) to confirm proper display and readability

---

## Quick Fix / Rollback

- [ ] Playlists → BFC HD → Edit; remove new slide(s) or move out of rotation
- [ ] Save and Apply Playlist; re-check kiosk display
