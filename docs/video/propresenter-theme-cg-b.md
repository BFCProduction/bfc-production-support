# Applying a ProPresenter Theme on CG B

CG B requires a specific ProPresenter theme applied to all song slides before each service. This page covers how to apply the correct theme.

---

## Overview

Song slides on CG B must use the **BFC Basic → 1 Line 1** theme. If the wrong theme is applied, song text will not display correctly on the confidence screens.

This step is completed during pre-service setup as part of the [Sunday Gameday Checklist](../workflows/sunday-gameday-checklist.md).

---

## Applying the Theme

Repeat this process for each song in the service playlist on CG B:

1. Open the song presentation in ProPresenter on CG B
2. Click the **first slide** to select it
3. Press **Cmd+A** to select all slides
4. Right-click any selected slide
5. Navigate to **Themes → BFC Basic → 1 Line**
6. Click to apply

The theme will be applied to all slides in the presentation.

---

## Notes

- The correct theme path is: **Themes → BFC Basic → 1 Line**
- Apply the theme to every song in the service — not just one
- If the theme does not appear, confirm the BFC Basic theme package is installed on CG B
- Prayer time slides on CG B should also be checked — see the Gameday Checklist
