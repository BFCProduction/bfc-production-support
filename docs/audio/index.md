# Audio

This section covers Dante network audio, mixing console setup, signal flow, and playback systems. If you're troubleshooting an audio issue or setting up a new routing configuration, start here.

---

## What's in This Section

- **Dante** — network audio routing, Dante Controller, device patching, and latency settings
- **Consoles** — setup, scene management, and show file procedures
- **Signal Flow** — how audio moves through the system from source to output
- **Playback** — click tracks, backing tracks, and playback system configuration
