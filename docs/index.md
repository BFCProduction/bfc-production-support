# Welcome to BFC Production Support

This is the internal knowledge base for the BFC Production team. It exists to capture how our systems work, how we've solved problems, and how we run repeatable processes — so that knowledge lives in the team, not just in someone's head.

#What You'll Find Here
##Audio 
— Dante network configuration, console setup, signal flow troubleshooting, and playback systems.
##Video 
— Resolume Arena workflows, codec and file format guides, display and signal troubleshooting.
##Lighting — GrandMA3 configuration, Art-Net and sACN setup, fixture patching, and console networking.
##Network 
— VLAN structure, switch configuration, IP addressing, and infrastructure references.
##Workflows 
— Standard operating procedures, repeatable checklists, and step-by-step processes for common tasks.

#How to Use This Site
Use the search bar at the top to find anything quickly. Use the left navigation to browse by category. Pages link to manufacturer documentation where relevant — those links are kept close to the procedures that need them.

#Contributing
If you've solved a problem, documented a process, or found a better way to do something — add it here. The goal is a knowledge base that gets more useful over time, not one that goes stale.
See Creating a New Support Page for instructions on how to add or edit content.
